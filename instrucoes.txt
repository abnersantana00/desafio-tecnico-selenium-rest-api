Parte (1) - Acessar o site da Prefeitura de Natal com Selenium 
 - Coletar todas as publicações referentes ao mês anterior
https://www.natal.rn.gov.br/dom
 - Para cada publicação baixe o documento em .PDF e armazene localmente


Parte (2) - Fazer Upload dos arquivos para o site https://0x0.st
- Após a coleta dos arquivos, fazer upload para o site:  https://0x0.st
Para fazer o upload basta realizar a requisição HTTP abaixo :
    Método : POST
    URL : https://0x0.st
    Body : multipart/form-data, campo file, contendo o arquivo a ser enviado
    Resposta: URL pública em texto puro (sem JSON)
Armazene as urls de upload de cada arquivo e dê um print na tela (isso eu faço)


Parte (3) - Armazenar em um Banco de Dados PostreSQL
- Após coletar as URLs públicas dos arquivos enviados no site https://0x0.st 
- Armazene as URLs públicas em um banco de dados PostgreSQL.
- Desenvovler uma API para essa inserção.

Parte (4) - Criar Rota na API de Acesso Público

- Desenvolva uma API que permita
    Listar todos os arquivos armazenados
    Filtrar os arquitov s por competência (mês da publicação)


Parte (5) - Publicação no GIT 
Disponibilize o código em um repositorio Git público, acompanhado de um arquivo README.md que explique de forma clara.


Passos extras:

Parte (6) - Dockerizar a API
- Crie um dockerfile para sua API e um arquivo docker-compose.yml que orquestra a API e o Banco de Dados.
(permita que sua aplicaçaõ seja executada em qualquer ambiente compatível com docker)

Parte (7) - Deploy do Projeto
- Como diferencial e um grande bônus realize o Deploy da API em um serviço de nuvem (AWS, Google Cloud, Azure, Heroku) ou em um PAAS (Railway, Render, Fly.io. Koyeb)
